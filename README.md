# College Event Page

A dynamic web application for displaying and managing college events at Northview College.

## Features

- **Event Display**: View upcoming college events with details including title, date, category, description, and location
- **Search Functionality**: Search events by title or date (YYYY-MM-DD format)
- **Category Filtering**: Filter events by category (Sports, Workshop, Guest Lecture)
- **Date Range Filter**: Filter events within a specific date range
- **Pagination**: Navigate through events with Previous/Next buttons (5 events per page)
- **Responsive Design**: Clean and modern UI with responsive layout

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Thiyanesh07/College-Event-Page.git
   ```

2. Open `index.html` in your web browser

3. Use the search bar to find events by title or date

4. Filter events by selecting a category from the dropdown

5. Set a date range to view events within specific dates

6. Navigate through pages using Previous/Next buttons
